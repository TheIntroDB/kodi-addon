Smart Intro Skip

Adding this site in Kodi as a source?

Use this folder URL so you only see the two zip files (nothing else):

https://jzonethegit.github.io/Smart-Intro-Skip/zips/

Then Settings → Add-ons → Install from zip file → open that source →

  1. Most people: install 01-install-this-first-repository.smartintro.jz-1.0.0.zip first. After that, use Add-ons → Install from repository → Smart Intro Skip repo to install the addon and get updates.
  2. Addon only (no repo): install plugin.video.introskip-1.0.1.zip instead.

Or download on a phone/PC and copy the zip to your device:

  • 01-install-this-first-repository.smartintro.jz-1.0.0.zip (repo — use first for updates)
  • plugin.video.introskip-1.0.1.zip (addon only)
